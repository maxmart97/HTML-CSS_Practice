Border-Style

The border style can a lot of different things: solid, dotted, dashed, double, groove, ridge, inset, outset, hidden, and none.
It can also be specified by using the border-[top, left, right, bottom]-style property.

Border-Color

The border-style can be changed to any color using any of the ways of choosing color. The border-style can also be specified using border-[top, left, right, bottom]-color or using border: color color color color.

Padding

Padding makes text closer or farther away from the border. It is usually specified in pixels. Padding can also be specified using padding-[top, right, bottom, left] or using padding: px px px px.

Margin

Margin controls the gaps between boxes. Specfied in px, em, or %. If width is specified, margin is added to the width of the box. If one box sits on top of another, larger of two paddings will be used. Can be specified using margin-[top, right, bottom, left], margin: px px px px, or margin px(top/bottom) px(left/right).

Centering Content

To center content inside of other boxes, you must set the width of the box you want centered. You then set the left and right margins to auto. If the width is not specified, the box will span the entire width of the box it is inside.

IE6 Box Model

Beware: IE6 includes padding and margin with width of box while newer browsers add padding and margin on top of width of box.

CSS3: Border-Image

A border image can be added to make custom borders. A border-width and border-style must be specified before a border-image will appear. The property requires 3 pieces of information:

  1. The url of the image: url("path/")
  2. Where to slice the image: # # # # or %
  3. What to do with the straight edges: [stretch, repeat, round]

CSS3: Box-Shadow

This property allows you to add a drop shadow around a box. It takes in: px px px px color. The px specification is the horizonal offset, vertical offset, blur distance, and spread of shadow. Horizontal and vertical offset must be specified. Inset can be placed at the beginning to create and inner-shadow.

CSS3: Border-Radius

The border radius property allows you to round corners. It can be specified using border-[top, bottom]-[right, left]-radius, or using border-radius: px px px px.

You can make weird shapes by adjusting the values to weird numbers. To make a circle, make radius the same height as a square box.
