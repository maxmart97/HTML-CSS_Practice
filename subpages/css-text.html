Attribute Selectors

There are other selectors that you can create rules with that apply to elements that have an attribute and a specific value.

  • Existence (targets all X elements with an attribute called XXXX): 
    [] -> Ex: element[attribute] -> p[class]
  • Equality (targets all X elements with an attribute called XXXX that has a value of XXXXXX): 
    [=] -> Ex: element[attribute="value"] -> p[class="animal"]
  • Space (targets all X elements with an attribute called XXXX that has a value of XXXXXX in a space-separated list of words): 
    [~=] -> Ex: element[attribute~="value"] -> p[class~="animal"]
  • Prefix (targets all X elements with an attribute called XXXX that has a value that starts with a specific string XXXXXX): 
    [^=] -> Ex: element[attribute^="value"] -> p[class^="ani"]
  • Substring (targets all X elements with an attribute called XXXX that has a value that contains substring XXXXXX): 
    [*=] -> Ex: element[attribute*="value"] -> p[class*="nim"]
  • Suffix (targets all X elements with an attribute called XXXX that has a value that ends with XXXXXX): 
    [$=] -> Ex: element[attribute$="value"] -> p[class$="al"]
