External CSS

To link to an external css stylesheet, put this in the head:

<link href="filepath" type="text/css" rel="stylesheet"/>

Internal CSS

To use internal css, you have multiple options:

  • Add to the head: 
    <style type="text/css">

    	selector {
    	   property: value;
    	}

    </style>
  • Add as an attribute to an element: 
    <p style="property: value;">Some text here</>

CSS Selectors

  • Universal Selector (Targets all elements on the page): 
    	* {}
  • Type Selector (Targets all elements of selected type): 
    	h1, h2, h3, p {}
  • Class Selector (Targets all elements of specific className or only specific elements with a specific className): 
    	.className {}, element.className {}
  • ID Selector (Targets element with specific ID): 
    	#idValue {}
  • Child Selector (Targets all occurrences of element2 that are children of element1) 
    	element1>element2 {}, ex: p>a {}
  • Descendant Selector (Targets any element2s that sit inside element1; element2s do not have to be direct children): 
    	element1 element2 {}, ex: p a {}
  • Adjacent Sibling Selector (Targets first occurrence of element2 after any occurrence of element1): 
    	element1+element2 {}, ex: h1+p {}
  • General Sibling Selector (Targets any element2s that are siblings of element1) 
    	element1~element2 {}, ex: h1~p {}

Cascading Rules

  • Identical selectors: Last selector takes precedence (color will be green not blue). 
    	i {
    	   color: blue;
    	}

    	i {
    	   color: green;
    	}
  • Specificity: The most specific selector takes precedence. Ex: h1 is more specific than *. If h1 has a rule that is the same as a rule inside of *, the rule that h1 follows is the one inside h1. 
    	* {
    	   color: blue;
    	}

    	h1 {
    	   color: green;
    	}
  • Importance: You can add "!important" after any value in a rule to say that this rule should be considered more important than other equal rules that apply to the same element. 
    	i {
    	   color: blue;
    	}

    	i {
    	   color: green !important;
    	}

Inheritance

All rules applied to parent elements also apply to their children elements unless overridden.
