DOCTYPES

There are many different versions of HTML. To specify the version you are using, you must include:
<!DOCTYPE html ...> at the beginning of the document. To see how to use other forms for html that are not HTML 5, see page 181 or look online.

Comments

To do a comment in HTML, use this: <!-- Comment text -->

ID and Class Attributes

The id and class attributes can be used on any HTML element. Id is specific to the one element it is inside and class is specific to all the elements that contain the class attribute. Both are used for styling purposes with CSS.
Ex: id="id_name" or class="class-name"

Block and Inline Elements

Block elements always start on a new line while inline elements stay on the line they are added to.
Examples:
Block - <h1>, <p>, <ul>, <li>
Inline - <a>, <b>, <em>, <img>

Div and Span Elements

Div elements allow you to group elements together into one box. Span elements allow you to group inline elements or text together on the same line. Both groups allow for easy styling of all the elements contained inside the group.
Div - <div>
Span - <span>

iFrames

Possible attributes: width="###", height="###", scrolling="yes,no,auto", frameborder="0,1", seamless="seamless"

Meta

The <meta> element is used to provide information about the page. It is not displayed on the screen. Meta elements are usually put inside the <head> element.
Attributes and values commonly used together (pgs 191-192):

  • name="description" content="the actual description of the page"
  • name="keywords" content="keyword, keyword, keyword"
  • name="robots" content="noindex/nofollow" (noindex means page will not be added to search engine results; nofollow means that the page will be added to search engine results but not any pages it links to)
  • http-equiv="author" content="the author's name"
  • http-equiv="pragma" content="no-cache" (prevents page from being cached by the browser)
  • http-equiv="expires" content="Fri, 02 Feb 2019 12:00:00 CST" (indicates when page expires and should no longer be cached)
